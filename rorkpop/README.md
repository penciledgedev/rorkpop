# RORK POP - WordPress Popup Plugin

A WordPress plugin that displays an email and data capture form popup, especially on video category pages.

## Description

RORK POP is designed to help collect visitor information through a stylish popup form. The plugin is specifically tailored for RORK TV, focusing on video content pages, but can be configured to display on all pages.

### Features

- Email and data capture form popup
- Targeted display on video category pages (configurable)
- Option to display on all pages
- Admin interface to view all submissions
- Export submissions as CSV
- Email verification
- Customizable logo

## Installation

1. Download the plugin zip file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New
4. Click "Upload Plugin" and select the zip file
5. Click "Install Now"
6. After installation, click "Activate Plugin"

## Usage

### Admin Settings

1. After activation, a new menu item "RORK POP" will appear in your WordPress admin menu
2. Navigate to RORK POP > Settings to configure the plugin
3. You can set whether to display the popup on all pages or just video category pages
4. You can provide a custom logo URL for the popup

### Viewing Submissions

1. Navigate to RORK POP > Submissions to view all form submissions
2. You can see the name, email, country, and submission date for each entry
3. Click "Export as CSV" to download all submissions in CSV format
4. You can delete individual submissions if needed
